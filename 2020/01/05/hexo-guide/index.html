使用github.io与hexo创建博客

发表于 2020-01-05
创建仓库
  • 创建名称为 [name].github.io 的仓库
安装及配置 Hexo
  • 安装 node.js
  • 安装 git
  • 安装 Hexo：npm install -g hexo-cli
  • 初始化：hexo init \hexo_init
本地预览
部署到 Github
  • 复制\hexo_init目录下的内容到[name].github.io仓库目录下
  • 修改根目录下_config.yml文件，添加仓库https地址
  • 安装hexo-deployer-git自动部署发布工具
  • 提交至 github
